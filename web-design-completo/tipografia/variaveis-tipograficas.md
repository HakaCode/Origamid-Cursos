# Variáveis Tipográficas

- Tamanho do Tipo
  - Ex: **14px**,**16px** ou **18px**.

- Altura da Linha
  - Ex: **1,5 x** Tamanho do Texto
  (1,5 X 14px = 21px)

- Largura do Texto
  - Máximo de 10/14 palavras por linha.

> ***OBS:*** Indicações feitas para corpo de texto utilizando tipografias como ***Helvetica*** e ***Arial***
