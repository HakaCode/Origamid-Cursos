# Tipografia

> "É a arte e processo de criação de um texto. O objetivo principal é dar ordem estrutural e forma a comunicação." ***─ Wikipédia***

> "Se a tipografia faz algum sentido, ele é visual e histórico." ***─ Robert Bringhurst***
